# sosohan_project

# 브랜치 설정하기
* `git clone https://github.com/kimyeheun/sosohan_project.git`
* `python -m venv venv`
* `source venv/Scripts/activate`
* `cd sosohan_project`  
* `git branch [브랜치 이름]`
* `git checkout [브랜치 이름]`
* `git push -u origin [브랜치 이름]`

# 가상환경 안에서 다운 받아야 할 것들
* `pip install django`
* `pip install pip --upgrade`
